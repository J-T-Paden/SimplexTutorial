Building Frames

  • Gather top and bottom end bars.
  • Gather both side bars.
  • Confirm all metal has correct dimensions and no imperfections such as scratches, dents, etc..
  • Using a rubber mallet, force the top and bottom end bars into the 2 side bars.
  • File each corner down until dull.
  • Using a corner jig and a center punch, mark each corner to be drilled for Type F screws.
  • Drill each corner with a 3/16" bit.
  • Use a straight shank, bore out the corner holes until the size of Type F screws.
  • Clean up holes with a countersink bit.
